# Orisa Casa de Beleza

Site promocional da Orisa Casa de Beleza, salao especializado em cabelos naturais e com curvatura, localizado na Pituba, Salvador-BA.

Acesse em: <https://orisa-casa-de-beleza.vercel.app>

---

## Tecnologias

- HTML5, CSS3, JavaScript (vanilla)
- Google Fonts (Playfair Display, Poppins)
- Font Awesome 6.5.1
- Google Maps Embed API

## Secoes do Site

1. **Navegacao** -- barra fixa com scroll suave
2. **Hero** -- apresentacao com logo e CTA para WhatsApp
3. **Sobre** -- historia do salao com carrossel de imagens
4. **Servicos** -- cards com os principais servicos oferecidos
5. **CTA** -- banner de agendamento via WhatsApp
6. **Depoimentos** -- avaliacoes de clientes
7. **Contato** -- endereco, telefone, mapa e horario de funcionamento
8. **Footer** -- links e informacoes complementares

## Funcionalidades

- Layout responsivo (desktop, tablet e mobile)
- Navbar com efeito de transparencia ao scroll
- Carrossel de imagens com autoplay e suporte a swipe
- Animacoes de entrada com IntersectionObserver
- Botao flutuante do WhatsApp
- Fallback local para imagens do Instagram
